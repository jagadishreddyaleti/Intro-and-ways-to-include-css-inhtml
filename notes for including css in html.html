ways to include css in html

  1. inline css
  2. internal css
  3. external css

inline css

when we use css in same line of html it is known as inline css

disadvantages

1.increse of code size
2.takes more to make changes at many places

internal css

when we use css in same file of html it is called as internal css
In internal css the disadvantages of inline css are solved because internal css is applied to all the lines in the page.

disadvantages

1.increse of code size when we have to make changes in multiple pages
2.takes more time to style when we have multiple pages to make changes

External css

when we use css for multiple pages at once with the help of extenal file linking all the pages of html to one external page it is called as external css
In external css the disadvantages of internal css are solved beacuse we can change styling in multiple pages at only one change in external file.
